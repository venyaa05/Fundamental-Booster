# Fundamental-Booster
# 🧑‍💻 Interactive Personal Data Collector

Welcome to the **Interactive Personal Data Collector** project! 🎉

This Python program collects basic user information and displays: - 👤
Name - 🎂 Age - 📏 Height - 🔢 Favourite Number - 🧠 Data Types - 🏷️
Memory Addresses - 📅 Approximate Birth Year

------------------------------------------------------------------------

## 🚀 Features

✨ Accepts user input interactively.

✨ Demonstrates Python data types using `type()`.

✨ Displays memory addresses using `id()`.

✨ Calculates the approximate birth year.

## 🛠️ Steps to Run

1.  📥 Download or clone this project.
2.  🐍 Make sure Python is installed.
3.  ▶️ Open a terminal in the project folder.
4.  📝 Run the program:

``` bash
python "Fundamental Booster.py"
```

5.  ⌨️ Enter the requested details.
6.  🎉 View the collected information and calculated birth year.
